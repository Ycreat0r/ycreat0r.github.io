# 🎬 Vladislav Portfolio

Modern cinematic portfolio website for video & photography.

Inspired by premium creative websites like Luma, Framer and Apple.

---

## Features

- Dark cinematic design
- Fullscreen hero video
- Responsive layout
- GSAP animations
- Smooth scrolling
- Portfolio filtering
- Search system
- Video/photo lightbox
- Mobile navigation
- Lazy loading
- GitHub Pages ready

---

## Technologies

- HTML5
- CSS3
- JavaScript
- GSAP
- Lenis Smooth Scroll
- Tailwind CDN
